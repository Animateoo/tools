Cómo instalar

Extensiones CEP (.zxp) o scripts ScriptUI (.jsx).

01

Descarga

Entra al repo de la tool y descarga el release (.zxp o .jsx).

02

Instala

CEP: arrastra el .zxp con ZXP Installer.
ScriptUI: copia el .jsx en la carpeta ScriptUI Panels de Scripts.

03

Abre el panel

AE/Pr: Window → Extensions → [Tool]
Scripts: Window → [Tool]
